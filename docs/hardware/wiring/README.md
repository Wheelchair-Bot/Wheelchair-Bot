# Wheelchair-bus wiring & pinouts

Per-family safe-tap procedures and pinouts.

> **Read this first.** Tapping a wheelchair control bus is electrically
> safe if done carefully but can void the OEM warranty and, if you bridge
> the wrong pins, can destroy the chair's controller. Always:
> 1. Disconnect the battery before opening any harness.
> 2. Buzz pin-to-pin with a meter before powering on.
> 3. Use isolated CAN transceivers (e.g. ISO1050) for the first prototype.
> 4. Keep the OEM joystick connected during all of Phase 3 (adapter mode).
